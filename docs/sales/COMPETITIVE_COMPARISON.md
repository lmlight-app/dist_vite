## LM Light の差別化ポイント

### 1. ワンコマンドインストール

他のソリューションはDocker環境が前提ですが、LM Light はDockerなしでもワンコマンドでインストール可能。ITリソースが限られた環境でも導入できます。

### 2. 業務特化機能

一般的なチャット・RAGに加え、SQLエージェント、承認フロー、DXF処理、物体検出など、業務に直結する機能を標準搭載しています。

### 3. エンタープライズ認証

Active Directory / Azure AD (Microsoft Entra ID) との連携に標準対応。既存のID管理基盤をそのまま利用でき、ユーザーは初回ログイン時に自動でアカウントが作成されます。

### 4. 買い切りライセンス

サブスクリプションだけでなく、一度の支払いで永続利用できる買い切りライセンスを選択可能。ランニングコストを抑えたい組織に最適です。

### 5. 完全日本語対応・国内サポート

UIは日本語ネイティブ。国内企業（デジタルベース株式会社）による直接サポートが受けられます。
