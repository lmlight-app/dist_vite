## 2. 製品概要

DigitalBase は、企業内にインストールして利用するセルフホスト型の AI データ連携基盤です。

ローカル AI（Ollama / vLLM）を活用し、社内データを外部に送信することなく、AI チャット・社内文書検索（RAG）・データ連携パイプライン（ETL）・業務承認フローなどの機能を提供します。

### 対応プロセス

| 種別 | Pコード | プロセス名 | DigitalBase の該当機能 |
|------|--------|-----------|---------------------|
| 業務プロセス | 共P-05 | 総務・人事・給与・労務・教育訓練・法務・情シス・統合業務 | ユーザー管理、認証連携（LDAP/OIDC）、承認フロー |
| 汎用プロセス | 汎P-07 | 業種・業務が限定されない生産性向上ツール | AIチャット、RAG、ETLパイプライン、SQLエージェント、ドキュメント生成 |

### 🔶 AI機能について

本ソフトウェアには以下の **AI技術** が搭載されています。

| AI分類 | 該当機能 | 説明 |
|--------|---------|------|
| **🔶 生成AI** | AIチャット、RAG、ドキュメント生成 | LLM（大規模言語モデル）による文章生成・質問応答・文書要約 |
| **🔶 生成AI以外のAI技術** | ETL AI変換、物体検出（YOLO）、OCR | データ分類・分析、画像認識、文字認識 |

### 導入により解決できる課題

- 社内文書・マニュアルの検索に時間がかかる → **RAG による AI 検索で即座に回答**
- 複数システムからのデータ集約に手作業が多い → **ETL パイプラインで自動化**
- クラウド AI にセキュリティ上のデータを送れない → **ローカル AI で完結**
- 部門ごとに AI ツールがバラバラ → **1つの基盤に統合、Active Directory 連携**

## 🔶 3. ITツールの利用方法

### 3.1 導入の流れ

```
① ライセンス購入（IT導入支援事業者を通じて購入）
    ↓
② インストール（ワンコマンド、所要時間 約5分）
   - Windows: PowerShellコマンド1行で自動インストール
   - macOS/Linux: シェルコマンド1行で自動インストール
   - データベース・初期設定も自動で完了
    ↓
③ 初期設定（管理者がブラウザからログイン）
   - デフォルトの管理者アカウントでログイン
   - 必要に応じてActive Directory / Azure AD認証を設定
    ↓
④ ユーザー追加・権限設定
   - 管理画面からユーザーを登録（またはAD連携で自動作成）
   - ADMIN / SUPER / USER のロールを割り当て
    ↓
⑤ 利用開始（ブラウザでアクセス）
```

### 3.2 日常の利用方法

1. **ブラウザでアクセス**: ユーザーは `http://<サーバーIP>:8000` にアクセス
2. **ログイン**: ID/パスワード（またはActive Directory / Azure AD）で認証
3. **機能選択**: 左メニューから利用したい機能を選択
   - **AIチャット**: 質問を入力 → AIが回答を生成
   - **RAG**: 文書をアップロード → AIが内容を学習 → 質問に回答
   - **ETLパイプライン**: ドラッグ&ドロップでデータ処理フローを構築 → スケジュール自動実行
   - **承認フロー**: 申請作成 → 上長に通知 → 承認/却下
   - **SQLエージェント**: 自然言語でデータベースに問い合わせ
   - **Webhookワークフロー**: 外部サービスとの連携・通知自動化
4. **管理者機能**: ダッシュボードからユーザー管理・設定変更・監査ログ確認・サイドバーカスタマイズ

### 3.3 利用に必要な環境

| 項目 | 要件 |
|------|------|
| サーバー | Windows 10+ / macOS 12+ / Linux（メモリ 8GB以上、16GB推奨） |
| クライアント | Webブラウザ（Chrome, Edge, Safari, Firefox） |
| ネットワーク | LAN内アクセス（インターネット不要。クラウドLLM利用時のみ必要） |
