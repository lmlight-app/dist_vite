## はじめに

デジタルベース株式会社（以下「当社」）は、LMLight（以下「本製品」）の提供にあたり、お客様の個人情報の保護を重要な責務と認識し、個人情報の保護に関する法律（以下「個人情報保護法」）およびその他関連法令を遵守し、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

## 第1条（収集する情報）

当社は、本製品の提供にあたり、以下の情報を収集することがあります。

### 1.1 お客様から直接提供いただく情報

| 情報 | 目的 |
|------|------|
| 会社名・担当者名 | 契約管理、サポート対応 |
| メールアドレス | ライセンス発行、連絡、サポート |
| 電話番号 | 緊急時の連絡 |
| 住所 | 請求書送付、契約書郵送 |
| 支払い情報 | 料金の請求・決済 |

### 1.2 ライセンス認証時に取得する情報

| 情報 | 目的 |
|------|------|
| Hardware UUID | ライセンス認証（デバイス紐付け） |

※ Hardware UUID はライセンス発行時にお客様から提出いただくものであり、本製品が自動的に外部へ送信することはありません。

### 1.3 収集しない情報

本製品はオンプレミス（お客様の環境内）で動作するため、以下の情報は当社に送信されません。

- チャット内容・会話履歴
- アップロードされたドキュメント
- RAGデータ・埋め込みベクトル
- ユーザーが作成したBot情報
- 文字起こし・物体検出の処理データ

**本製品で処理されるすべてのデータは、お客様の環境内に留まります。**

## 第3条（第三者提供）

1. 当社は、以下の場合を除き、お客様の個人情報を第三者に提供しません。
   - お客様の同意がある場合
   - 法令に基づく場合
   - 人の生命、身体または財産の保護のために必要がある場合
   - 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
2. 当社は、業務の一部を外部に委託する場合、委託先に対して適切な監督を行います。

## 第4条（安全管理措置）

当社は、個人情報の漏洩、滅失、毀損を防止するため、以下の安全管理措置を講じます。

### 組織的安全管理措置
- 個人情報管理責任者の設置
- 個人情報の取扱いに関する規程の整備・運用

### 人的安全管理措置
- 従業者との秘密保持契約の締結
- 個人情報の取扱いに関する教育・研修の実施

### 物理的安全管理措置
- 個人情報を取り扱う区域の管理
- 機器および電子媒体の盗難防止措置

### 技術的安全管理措置
- アクセス制御の実施
- 通信経路の暗号化
- 不正アクセスの検知・防止
