## 製品サマリー

**LM Light** は、完全オンプレミスで動作するLLMチャット・RAGプラットフォームです。

### 主要機能（13機能）

| 機能 | 説明 |
|------|------|
| AIチャット | 複数LLMモデルの切替・マルチユーザー |
| RAG | 社内文書をAIが検索・回答（PDF/Word/Excel等対応） |
| ドキュメント生成 | PDF・画像からテーブル・DXF等を自動生成 |
| SQLエージェント | 外部DBに自然言語でクエリ |
| 業務フロー連携 | Webhookベースのワークフロー |
| 承認フロー | 多段階承認プロセス・通知 |
| 文字起こし | 音声→テキスト変換（Whisper） |
| 画像処理 / Vision | 物体検出（YOLO）・DXF処理 |
| ベンチマーク | LLMモデルの性能比較 |
| プロンプトライブラリ | プロンプトの保存・共有 |
| ファインチューニングガイド | 学習データテンプレート |
| ブランディング | カスタムロゴ・タイトル・カラー |
| ユーザー管理 | 3段階ロール（ADMIN/SUPER/USER）+ タグ共有 |

### セキュリティ・認証

| 項目 | 内容 |
|------|------|
| データ保存 | お客様環境内のみ（外部送信なし） |
| 認証方式 | ローカル / **LDAP（Active Directory）** / **OIDC（Azure AD）** |
| パスワード | bcrypt（12ラウンド） |
| セッション | JWT + HTTP-only Cookie |
| LLM通信 | httpxでOpenAI互換エンドポイントに直接リクエスト（OpenAI SDK不使用） |
| ライセンス紐付け | 購入メールアドレス + Hardware UUID |

### 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Next.js 15 + React 19 + Prisma |
| バックエンド | FastAPI（Python）+ SQLAlchemy |
| データベース | PostgreSQL 17 + pgvector |
| LLMエンジン | Ollama（macOS/Linux/Windows）/ vLLM（Linux） |
| 認証 | NextAuth v5 + ldapts + OIDC |
