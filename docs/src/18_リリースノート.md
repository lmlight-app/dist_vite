# DigitalBase リリースノート

**Release Notes**

---

## v1.0.0 — 2026年4月1日

### 新機能

- **AIチャット**: 複数LLMモデルの切替・会話履歴管理・ストリーミング応答
- **クラウドLLM連携**: OpenAI (GPT-4o等) / Anthropic (Claude等) / Google (Gemini等) に対応
- **Tool Calling**: LLMからの関数呼び出しによる外部ツール連携
- **Web検索RAG**: Web検索結果をコンテキストに含めたAI回答生成
- **RAG**: 社内文書アップロード・ベクトル検索による回答生成（PDF/Word/Excel/PPT/CSV/JSON/画像対応）
- **ETLパイプライン**: データ取込 → AI変換（map/filter/reduce） → 出力の自動化
- **パイプラインオペレーター**: map / filter / reduce / sort / dedupe 等のデータ処理演算子
- **SQLエージェント**: 外部データベースへの自然言語クエリ
- **ドキュメント生成**: PDF・画像からテーブル・DXF等を生成
- **承認フロー**: 多段階承認プロセス
- **ヘルプデスク**: 社内問い合わせ管理・ナレッジベース連携
- **文字起こし**: Whisperによる音声ファイルのテキスト変換（WAV/MP3/M4A/MP4/WebM等）
- **画像処理 / Vision**: YOLOv8物体検出・DXF処理
- **認証**: ローカル認証 / LDAP (Active Directory) / OIDC (Azure AD / Microsoft Entra ID)
- **ユーザー管理**: 3段階ロール（ADMIN/SUPER/USER）・タグベースのアクセス制御
- **ブランディング**: ロゴ・テーマカラーのカスタマイズ
- **ベンチマーク**: LLMモデルの性能比較ツール
- **LLMエンジン**: Ollama版 / vLLM版（httpxによるOpenAI互換API通信）
- **対応OS**: macOS / Linux / Windows（Docker版含む）
- **ワンコマンドインストール**: `db start` / `db-vllm start` で即座に起動

### 破壊的変更
- 製品名を「LM Light」から「DigitalBase」に変更
- 起動コマンドを `lmlight start` から `db start` に変更
- インストールディレクトリを `~/.local/lmlight` から `~/.local/db` に変更
