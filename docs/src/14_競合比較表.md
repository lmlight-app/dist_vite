## DigitalBase の差別化ポイント

### 1. ワンコマンドインストール

他のソリューションはDocker環境が前提ですが、DigitalBase はDockerなしでもワンコマンドでインストール可能。ITリソースが限られた環境でも導入できます。

### 2. ETLパイプライン & AI Transform

n8nのようなワークフローツールと異なり、DigitalBase はパイプラインオペレーター（map/filter/reduce等）とローカルLLMによるAI変換を統合。データの取込・加工・分析をひとつのプラットフォームで完結できます。

### 3. ローカルLLM + クラウドLLM ハイブリッド

Ollama / vLLM によるローカル推論に加え、OpenAI・Anthropic・Google のクラウドLLMにも対応。用途やコストに応じて使い分けが可能です。

### 4. 業務特化機能

一般的なチャット・RAGに加え、SQLエージェント、承認フロー、ヘルプデスク、DXF処理、物体検出、Web検索RAG、Tool Callingなど、業務に直結する機能を標準搭載しています。

### 5. エンタープライズ認証

Active Directory / Azure AD (Microsoft Entra ID) との連携に標準対応。既存のID管理基盤をそのまま利用でき、ユーザーは初回ログイン時に自動でアカウントが作成されます。

### 6. 買い切りライセンス

サブスクリプションだけでなく、一度の支払いで永続利用できる買い切りライセンスを選択可能。ランニングコストを抑えたい組織に最適です。

### 7. 完全日本語対応・国内サポート

UIは日本語ネイティブ。国内企業（デジタルベース株式会社）による直接サポートが受けられます。
