## 3. ご提案内容

### 3.1 製品概要

DigitalBase は、オンプレミス環境で動作する LLM チャット・RAG・業務自動化プラットフォームです。社内データを外部に出さずに AI を活用でき、Pipeline（80以上のオペレータ）、MCP サーバー連携、Helpdesk、Document Creator、Web 検索 RAG、クラウド LLM 併用にも対応しています。

### 3.2 導入構成

| 項目 | 内容 |
|------|------|
| エディション | ☐ Ollama版 / ☐ vLLM版 |
| ライセンス | ☐ 買い切り / ☐ サブスクリプション |
| ライセンス数 | ＿＿ライセンス |
| サポートプラン | ☐ スタンダード / ☐ プレミアム / ☐ エンタープライズ |

|------|------|------|

| Web 検索 RAG | ☐ | |

## 4. システム構成

### 4.1 必要環境

| 項目 | 推奨スペック |
|------|------------|
| OS | ＿＿＿＿ |

### 4.2 ネットワーク

| 項目 | 内容 |
|------|------|
| 使用ポート | API + Web: 8000（FastAPI が SPA 同居配信、単一ポート）, DB: 5432, Ollama: 11434（vLLM版: 8080/8081） |

| バインド | デフォルト `API_HOST=0.0.0.0`（LAN 露出）。ハードニング時は `127.0.0.1` |
| アクセス方式 | ☐ ローカルのみ / ☐ LAN内共有 |

## 5. セキュリティ

| 項目 | 内容 |
|------|------|
| データ保存場所 | お客様環境内（既定で外部送信なし） |
| 認証方式 | ID/パスワード（passlib bcrypt）/ LDAP（python-ldap3）/ OIDC（python-jose） |
| アクセス制御 | ロールベース（ADMIN/SUPER/USER）+ タグ ACL + runAs（caller / owner）+ LDAP グループ → タグ自動マッピング |
| セッション管理 | JWT (HS256) + HTTP-only Cookie |
| 暗号化 | 通信経路のHTTPS対応可（Nginx リバースプロキシ） |
