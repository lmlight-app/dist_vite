### 事例1: 製造業（大和製罐株式会社） — Visionモデルによる製造現場のAI活用

| 項目 | 内容 |
|------|------|
| 取引先 | 大和製罐株式会社 |
| 業種 | 製造業 |
| 利用エディション | Ollama版 |
| 導入プラン | 永年買い切り スタンダード |

**課題:**
- 製造現場の画像データ（製品写真・図面等）の分析に人手がかかっていた
- 外部クラウドAIにセキュリティ上、製造データをアップロードできなかった
- 画像を含む社内資料の検索・活用が非効率だった

**導入した機能:**
- Vision モデル（画像認識・画像からのテキスト抽出）
- AIチャット（Visionモデルによる画像付き質問応答）
- ドキュメント生成（図面・画像からのデータ抽出）

**導入効果:**
- Visionモデルにより製造現場の画像データをAIで分析可能に
- ローカル環境で完結するため機密性の高い製造データも安全に処理
- 画像を含む業務のAI化により作業効率が向上

### 事例2: 国立大学法人（大阪大学） — RAGと承認機能による学内業務効率化

| 項目 | 内容 |
|------|------|
| 取引先 | 国立大学法人大阪大学 |
| 業種 | 教育・研究機関 |
| 利用エディション | Ollama版 |
| 導入プラン | 永年買い切り スタンダード |

**課題:**
- 学内の規程・マニュアル・研究資料が分散しており、検索に時間がかかっていた
- 申請・承認プロセスが紙ベースで非効率だった
- 機密性の高い研究データを外部クラウドに送れなかった

**導入した機能:**
- RAG（学内規程・研究資料のAI検索）
- 承認フロー（学内申請の多段階承認プロセス・通知）
- AIチャット（学内業務に関する質問応答）

**導入効果:**
- RAGにより学内資料の検索時間を大幅に短縮
- 承認フローのデジタル化により申請・承認プロセスを効率化
- 学内データの機密性を保持したままAI活用を実現
