## 事例1: 大和製罐株式会社（製造業） — Visionモデルによる製造現場のAI活用

### 企業プロフィール

| 項目 | 内容 |
|------|------|
| 取引先 | 大和製罐株式会社 |
| 業種 | 製造業 |
| 利用エディション | Ollama版 |
| 導入プラン | 永年買い切り スタンダード |

### 課題

- 製造現場の画像データ（製品写真・図面等）の分析に人手がかかっていた
- 外部クラウドAIにセキュリティ上、製造データをアップロードできなかった
- 画像を含む社内資料の検索・活用が非効率だった

### 導入した機能

| 機能 | 用途 |
|------|------|
| Vision モデル | 製造現場の画像データの認識・テキスト抽出 |
| AIチャット | Visionモデルによる画像付き質問応答 |
| ドキュメント生成 | 図面・画像からのデータ抽出 |

### 導入効果

- Visionモデルにより製造現場の画像データをAIで分析可能に
- ローカル環境で完結するため機密性の高い製造データも安全に処理
- 画像を含む業務のAI化により作業効率が向上

## 事例2: 国立大学法人大阪大学（教育・研究機関） — RAGと承認機能による学内業務効率化

### 企業プロフィール

| 項目 | 内容 |
|------|------|
| 取引先 | 国立大学法人大阪大学 |
| 業種 | 教育・研究機関 |
| 利用エディション | Ollama版 |
| 導入プラン | 永年買い切り スタンダード |

### 課題

- 学内の規程・マニュアル・研究資料が分散しており、検索に時間がかかっていた
- 申請・承認プロセスが紙ベースで非効率だった
- 機密性の高い研究データを外部クラウドに送れなかった

### 導入した機能

| 機能 | 用途 |
|------|------|
| RAG | 学内規程・研究資料のAI検索 |
| 承認フロー | 学内申請の多段階承認プロセス・通知 |
| AIチャット | 学内業務に関する質問応答 |

### 導入効果

- RAGにより学内資料の検索時間を大幅に短縮
- 承認フローのデジタル化により申請・承認プロセスを効率化
- 学内データの機密性を保持したままAI活用を実現

## 事例3: Spingence Technology Co., Ltd.（情報通信業・台湾） — マルチGPU環境でのAIプラットフォーム稼働

### 企業プロフィール

| 項目 | 内容 |
|------|------|
| 取引先 | Spingence Technology Co., Ltd. |
| 業種 | 情報通信業（AIインフラ） |
| 所在地 | 台湾・台北市 |
| 利用エディション | vLLM版（NVIDIA マルチGPU環境） |

### 概要

- 台湾のAIインフラ企業と共同で、マルチGPU環境上で稼働するAIデータ連携プラットフォームを構築
- DigitalBase の vLLM版をベースに、複数のNVIDIA GPUを活用した高スループットなAI推論基盤を実現
- 大規模言語モデルの並列処理により、多数の同時利用ユーザーへの高速なAI応答を提供

### 導入した機能

| 機能 | 用途 |
|------|------|
| vLLM 連携 | マルチGPU環境での高スループット LLM 推論 |
| AIチャット | 多数同時ユーザー対応のAIアシスタント |
| ETLパイプライン | データ連携・自動処理基盤 |

### 導入効果

- マルチGPU環境を活用した高速 AI 推論基盤を構築
- アジア圏での DigitalBase 海外展開実績
- 大規模言語モデルの並列処理による多ユーザー同時利用への対応
