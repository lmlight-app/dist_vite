# サービスレベル合意書（SLA）

**Service Level Agreement**

最終更新日: 2026年3月

---

## 第1条（目的）

本サービスレベル合意書（以下「本SLA」）は、デジタルベース株式会社（以下「甲」）が提供するLMLight（以下「本製品」）に関するサポートおよびサービスの品質基準を定めるものです。

---

## 第2条（サポート範囲）

### 2.1 サポート対象

| 対象 | 内容 |
|------|------|
| インストール | 本製品のインストール・初期設定に関する支援 |
| 設定 | 環境変数、ネットワーク設定等の設定支援 |
| 障害対応 | 本製品の不具合に起因する障害の調査・対応 |
| アップデート | 本製品のアップデート手順の案内・支援 |
| ライセンス | ライセンスの発行・認証に関する対応 |

### 2.2 サポート対象外

| 対象外 | 説明 |
|--------|------|
| ハードウェア障害 | サーバー・PC自体の故障 |
| OS・ネットワーク | お客様環境のOS設定、ネットワーク構築 |
| カスタマイズ | 本製品の改変・カスタム開発 |
| サードパーティ製品 | Ollama、PostgreSQL等の個別サポート |
| データ復旧 | お客様のデータバックアップ・復旧 |

## 第4条（障害の優先度）

| 優先度 | 定義 | 対応目標 |
|--------|------|---------|
| **P1 - 緊急** | 本製品が完全に使用不能。業務に重大な影響 | 4時間以内に調査開始 |
| **P2 - 高** | 主要機能が使用不能だが、回避策あり | 1営業日以内に調査開始 |
| **P3 - 中** | 一部機能の不具合。業務への影響は限定的 | 3営業日以内に調査開始 |
| **P4 - 低** | 軽微な不具合、改善要望 | 次回アップデートで検討 |

### 5.1 アップデートの種類

| 種類 | 内容 | 提供方法 |
|------|------|---------|

## 第6条（除外事項）

以下の場合、本SLAに基づくサポートの対象外とします。

1. お客様による本製品の改変に起因する問題
2. 当社が推奨しない環境での使用に起因する問題
3. 第三者製品（OS、ハードウェア、ネットワーク機器等）に起因する問題
4. 天災、停電、通信障害等の不可抗力に起因する問題
5. お客様の故意または重過失に起因する問題

2. 変更に同意できない場合、お客様は変更の効力発生日前に書面にて異議を申し立てることができます。
